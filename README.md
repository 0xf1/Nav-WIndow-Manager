# Nav-Window-Manager
Менеджер окон Windows клиента Business Central / Dynamics Nav.

Функции:
* Переключение на нужное окно (двойной клик)
* Сортировка окон, процессов
* Закрытие выделенных окон
* Обнаружение модальных окон (когда Windows клиент не дает работать с окнами, так как запущена модальная страница/диалог)

![alt text](https://github.com/0xf1/Nav-WIndow-Manager/blob/main/pic2_.JPG?raw=true)

Дополнительно:
* Окно *Settings/Arg transformers...* служит для замены аргументов коммандной строки на форме на читаемое значение
